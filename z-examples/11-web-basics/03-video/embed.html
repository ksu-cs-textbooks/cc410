Java

  1. Run project once to see what it does. Explore code and show JSON loader and Movie class.
  2. Go to start.spring.io and get a build.gradle file from there. Use it to update the one in the starter project.
  3. Create movies.web package.
  4. Create Web.java in movies and MoviesController.java in movies.web.
  5. Copy code from WebApplication.java file in Spring Initializr to Web.java and adjust
  6. Set main class in build.gradle to movies.Web
  7. Create controller in MoviesController.java and configure home page.
  8. Create resources folder in the src/main directory. Add a static and templates folder inside.
  9. Create an index.html file inside of the templates directory.
  10. Run project and see how it looks. gradle bootRun
    1. Configure the .codio file to add a new tab at the top. Port 8080!
  11. Create layout file layout.html as a basic layout.
  12. Update index.html to use layout.
  13. Test!
  14. Add greeting paths to Controller & create greeting template.
  15. Test!
  16. Update index.html route to list movies in a UL. Test!
  17. Update index.html to have more HTML and data.
  18. Update index.html to have CSS classes and then add CSS file to Layout. It goes in static folder.
  19. Add ratings and more CSS.
  20. Add Bootstrap stuff.

Python

  1. Run project once to see what it does. Explore code and show JSON loader and Movie class.
  2. Add flask and flask-classful dependencies and install. Remind to use tox -r once.
  3. Create movies.web package.
  4. Create Web.py in movies and MoviesController.py in movies.web.
  5. Build Web.py to launch Flask app
  6. Set main class in __main__.py to movies.Web and update __init__.py to load the app.
    1. Configure the app using export FLASK_APP="src"
  7. Create controller in MoviesController.java and configure home page.
  8. Create a static and templates folders inside src/movies directory.
  9. Create an index.html file inside of the templates directory.
  10. Run project and see how it looks. python3 -m flask run --host=0.0.0.0
    1. Configure the .codio file to add a new tab at the top. Port 5000!
    2. Install python-dotenv and create .flaskenv file
  11. Create layout file layout.html as a basic layout.
  12. Update index.html to use layout.
  13. Test!
  14. Add greeting paths to Controller & create greeting template.
  15. Test!
  16. Update index.html route to list movies in a UL. Test!
  17. Update index.html to have more HTML and data.
  18. Update index.html to have CSS classes and then add CSS file to Layout. It goes in static folder.
  19. Add ratings and more CSS.
  20. Add Bootstrap stuff.
  21. REMIND THEM TO MAKE about page! Play around with Bootstrap stuff!
